# 파이썬 챗봇



파이썬은 3가지 역할을 수행한다.

| 저장 | 조건 | 반복 |
| ---- | ---- | ---- |



1. 저장

   ```python
   a = 30
   b = '안녕'
   ```

   + `=`은 저장용 기호이다

     + 변수에 넣어주고, 입력하고, 저장하는 의미

   + `==`는 동일의 의미

     + ex.

       + ```python
         a == 3
         # a는 3이다. 참? 거짓?
         ```

2. 조건

   **boolean 연산자**

   | True(진실) | False(거짓) |
   | ---------- | ----------- |
   | 1          | 0           |

   **`and / &`** 연산자

   | a    | b    | 결과 |
   | ---- | ---- | ---- |
   | T    | T    | T    |
   | T    | F    | F    |
   | F    | T    | F    |
   | F    | F    | F    |

   **`or / |`** 연산자

   | a    | b    | 결과 |
   | ---- | ---- | ---- |
   | T    | T    | T    |
   | T    | F    | T    |
   | F    | T    | T    |
   | F    | F    | F    |

3.  list

   + 변수명 = []

     + ex

     + ```python
       num = [1,2,3]
       name = ['가','나','다']
       ```

4. 반복

   + `enumerate`

     + ex

     + ```python
       name = ['가','나','다','라']
       
       for idx, n in enumerate(name):
           print(f'순서 {idx+1}은 {n} 입니다.')
       ```

# CLI 기초

+ `pwd` : 현재 경로 확인
+ `ls` : 현재 폴더 파일 확인
+ `cd 경로` : 경로로 이동
+ `cd ..` : 상위 폴더로 이동
  + 경로속 경로로 이동시 `/` 사용
+ `clear` : 짜잔 내가 사라져볼게
+ `mkdir 이름` : 이름 폴더 생성
+ `touch 이름` : 이름 파일 생성(확장자 필수)
  + `vi 이름` : 파일 수정 (없을시 생성)
    + `i / Insert` : 문서 작성
    + `esc` : 취소
    + `:wq` : 저장후 파일닫기
+ `rm 이름` : 이름 파일 제거
+ `rm -r 이름` : 이름 폴더 제거
+ `mv 이름1 이름2` : 이름1 파일 이름2 로 변경
+ `mv 파일 폴더` : 파일 폴더로 이동



### 단축키

+ `위, 아래 방향기` : 과거에 작성했던 명령어 부르기!
+ `tab` : 과거에 작성했던 명령어 조회
+ `Ctrl + a` : 커서가 맨 앞으로 이동
+ `Ctrl + e` : 커서가 맨 뒤로 이동
+ `Ctrl + w` : 커서가 맨 앞 단어를 삭제
+ `Ctrl +l` : 터미널 화면을 깨끗하게 청소
+ `Ctrl + insert` : 복사
+ `Shift + insert` : 붙여넣기
